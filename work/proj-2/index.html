DrupalCamp Bangalore Website: A Drupal 7 website. I volunteered with the awesome Drupal Bangalore Community members and was involved in the frontend development of the site as well as designing the page elements ( Logo, Banners, Brochure, etc).

Tools and Technologies used

  • Mockup : Balsamiq Mockups

  • Graphics Design

    • Logo, Banner : Adobe Illustrator

    • Icons, Photos : Adobe Photoshop

    • Brochure : Adobe InDesign

  • Drupal Theming using Omega 4 Responsive Theme.

  • IDE : PHPStorm

  • VCS : GIT ( Acquia Cloud, Github )
